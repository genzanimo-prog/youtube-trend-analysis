# YouTube Trending Video Analysis
## What Drives Views and Engagement?

### Overview
Analysis of 40,949 US YouTube trending videos to identify patterns 
in views, engagement, and content strategy.

### Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook

### Key Findings
1. **Music** has the highest average views per video despite 
   Entertainment having the most trending videos
2. **Friday** is the best day to publish for maximum views
3. **Strong positive correlation** exists between likes and comments

### Files
- `youtube_analysis.ipynb` — Full analysis notebook
- `chart1_avg_views_by_category.png` — Views by category
- `chart2_views_by_publish_day.png` — Views by publish day
